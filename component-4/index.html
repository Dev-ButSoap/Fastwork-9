NOTE

1. Cutting Size: 90x55mm

นี้คือขนาดของชิ้นงานหลังจากทำการตัดเสร็จแล้ว

2. Safty Area - 3mm (inside)

ส่วนประกอบต่างๆที่อยู่นอกจากเส้นปลอดภัยอาจจะมีความเสี่ยงที่โดนตัดทิ้งออกไป เนื่องจากในกระบวนการผลิตอาจจะมีความคลาดเคลื่อนได้บ้างในบางครั้ง

3. Working size: 94x59mm

เป็นขนาดที่ไฟล์ที่ใช้ในการพิมพ์โดย Working size = cutting size + ระยะเผื่อตัดตกรอบด้าน 2mm เส้น Bleed คือส่วนที่จะถูกตัดออก ต้องปูพื้นหลังให้เต็มพื้นที่นี้เพื่อจะไม่ให้เห็นขอบขาวเวลาตัด เนื่องจากปกติการตัดจะมีความคลาดเคลื่อน 1-2 มม.

ปกติการผลิตอาจจะมีความคลาดเคลื่อนในการพิมพ์หรือตัด 1-2 มม. ดังนั้นจึงจำเป็นต้องต้องปูพื้นหลังให้เต็มพื้นที่ถึงขอบ เพื่อจะไม่ให้เห็นขอบขาวเวลาตัด (Working size = cutting size + 2mm margin bleed on all sides)

สำหรับผลลัพธ์ที่ดีในการพิมพ์ เราแนะนำให้ใช้ภาพที่มีความละเอียดสูงไม่ต่ำกว่า 300 dpi ถ้าหากต้องการคุณสามารถพิมพ์ภาพความละเอียดต่ำที่มี แต่ผลลัพธ์ก็จะเป็นตามคุณภาพของไฟล์

เนื่องจากข้อจำกัดของการแสดงผลจอภาพแต่จะเครื่องจะไม่เหมือนกัน เช่นความมืดสว่างเฉดสีเป็นต้น ดังนั้นสีงานพิมพ์ที่คุณเห็นจะไม่สามารถเทียบกับจอภาพได้ การสั่งผลิตสินค้าคนละรอบอาจจะมีความคลาดเคลื่อนของสีที่แตกต่างกันประมาณ 5-10% เนื่องจากปัจจัยหลายๆอย่างเช่นอุณหภูมิ ความชื่นเป็นต้น
